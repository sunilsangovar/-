Add New Tool Guide

Contributing to Advanced Tools

This guide will help you understand the process of adding a new tool to the Advanced Tools website. By following these steps, you can ensure your tool integrates seamlessly with the existing structure.

1. Understanding the Project Structure

The website follows a specific directory structure:

/
├── index.html              # Main homepage
├── css/
│   ├── style.css           # Main stylesheet
│   ├── themes.css          # Theme styles
│   └── tool.css            # Tool-specific styles
├── js/
│   └── main.js             # Main JavaScript file
├── tools/                  # Individual tool pages
│   ├── text-analyzer.html
│   ├── password-generator.html
│   └── [other-tool].html
├── pages/                  # Information pages
│   ├── about.html
│   ├── contact.html
│   └── privacy.html
└── [other pages].html

All tool-specific pages should be placed in the tools/ directory.

2. Creating a New Tool Page

Step 1: Create a new HTML file

Create a new HTML file in the tools/ directory. Use kebab-case for the filename (e.g., your-tool-name.html).

Step 2: Use the template structure

Copy the structure from an existing tool page and modify it for your new tool. Here's the basic template:

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>[Your Tool Name] - Advanced Tools Website</title>
    <link rel="stylesheet" href="../css/style.css">
    <link rel="stylesheet" href="../css/themes.css">
    <link rel="stylesheet" href="../css/tool.css">
    <link rel="stylesheet" href="https://cdnjs.cloudflare.com/ajax/libs/font-awesome/6.4.0/css/all.min.css">
</head>
<body class="theme-default">
    <!-- Header Section -->
    <header>
        <!-- Header content (same as other pages) -->
    </header>
    
    <!-- Main Content -->
    <div class="container">
        <!-- Sidebar -->
        <aside class="sidebar active" id="sidebar">
            <!-- Sidebar content (add your tool in the list) -->
        </aside>
        
        <!-- Content Wrapper -->
        <div class="content-wrapper">
            <div class="breadcrumb" id="breadcrumb">
                <a href="/index.html">Home</a>
                <span class="separator">></span>
                <span class="current-page">[Your Tool Name]</span>
            </div>
            
            <main class="content" id="main-content">
                <!-- Your tool content goes here -->
            </main>
        </div>
    </div>
    
    <!-- Footer -->
    <footer>
        <!-- Footer content (same as other pages) -->
    </footer>
    
    <!-- Scripts -->
    <script src="../js/main.js"></script>
    <!-- Add your tool-specific script here if needed -->
</body>
</html>
Make sure to update paths to CSS and JS files with ../ since tool pages are in a subdirectory.

3. Developing Your Tool Interface

Your tool's main content should include these sections:

Tool Header Section

<div class="tool-header">
    <h1>[Your Tool Name]</h1>
    <p class="tool-description">
        A brief description of what your tool does and its benefits to users.
    </p>
</div>

Tool Content Section

<div class="tool-container">
    <!-- Input Section -->
    <div class="tool-section">
        <h2>Input</h2>
        <!-- Your tool's input controls go here -->
        <div class="form-group">
            <!-- Example: -->
            <label for="user-input">Enter your data:</label>
            <input type="text" id="user-input" class="form-control">
        </div>
        <button id="process-btn" class="btn-primary">
            <i class="fas fa-cog"></i> Process
        </button>
    </div>
    
    <!-- Output Section -->
    <div class="tool-section">
        <h2>Result</h2>
        <div id="result-container">
            <!-- Results will be displayed here -->
        </div>
    </div>
</div>

Instructions Section (Optional)

<div class="tool-instructions">
    <h2>How to Use This Tool</h2>
    <ol>
        <li>First step instructions...</li>
        <li>Second step instructions...</li>
        <li>Third step instructions...</li>
    </ol>
</div>

4. Adding JavaScript Functionality

Create a separate JavaScript file for your tool in a js/tools/ directory (create it if it doesn't exist):

// js/tools/your-tool-name.js

document.addEventListener('DOMContentLoaded', function() {
    // Get elements
    const processBtn = document.getElementById('process-btn');
    const userInput = document.getElementById('user-input');
    const resultContainer = document.getElementById('result-container');
    
    // Add event listener
    processBtn.addEventListener('click', function() {
        // Get input value
        const input = userInput.value.trim();
        
        if (!input) {
            showError('Please enter some input');
            return;
        }
        
        // Process the data
        const result = processData(input);
        
        // Display the result
        displayResult(result);
    });
    
    // Helper functions
    function processData(input) {
        // Your tool's core functionality goes here
        return 'Processed: ' + input;
    }
    
    function displayResult(result) {
        resultContainer.innerHTML = `
            

                
Output

                
${result}

                
            

        `;
    }
    
    function showError(message) {
        resultContainer.innerHTML = `
            

                
                
${message}

            

        `;
    }
    
    // Additional helper functions as needed
});

// Function to copy result to clipboard
function copyToClipboard() {
    // Implementation here
}

Then include your script in your HTML file:

<script src="../js/tools/your-tool-name.js"></script>

5. Adding CSS Styles

Most styles are already defined in the main CSS files. If you need tool-specific styles, add them to a new CSS file:

/* css/tools/your-tool-name.css */

.your-custom-class {
    /* Your custom styles */
}

/* Add any tool-specific styles here */

Then include your CSS file in your HTML:

<link rel="stylesheet" href="../css/tools/your-tool-name.css">
Try to use existing CSS classes from the main stylesheet whenever possible to maintain consistent styling.

6. Updating the Website

Step 1: Add to Sidebar Navigation

Add your tool to the sidebar navigation in your tool page and all other pages:

<li><a href="tools/your-tool-name.html"><i class="fas fa-[appropriate-icon]"></i><span>Your Tool Name</span></a></li>

Step 2: Add to Homepage

Add your tool to the appropriate section in index.html:

<a href="tools/your-tool-name.html" class="tool-card">
    <div class="tool-icon"><i class="fas fa-[appropriate-icon]"></i></div>
    <div class="tool-details">
        <h3>Your Tool Name</h3>
        <p>Brief description of your tool</p>
    </div>
</a>

Step 3: Update Search Tools

Add your tool to the search tools array in js/main.js:

const tools = [
    // Existing tools...
    { 
        name: "Your Tool Name", 
        url: "tools/your-tool-name.html", 
        icon: "fas fa-[appropriate-icon]", 
        description: "Brief description of your tool" 
    },
];

7. Testing Your Tool

Before submitting your tool, ensure it passes these tests:

  • The tool loads correctly in multiple browsers
  • All functionality works as expected
  • The page is responsive on both desktop and mobile
  • The tool works in all themes and dark mode
  • Navigation and search functionality includes your tool correctly
  • The tool has proper error handling

8. Submission Process

Once your tool is ready, follow these steps to submit it:

  1. Check that all files are named correctly and placed in the proper directories
  2. Verify that all links and paths are correct
  3. Make sure your code follows the project's coding standards
  4. Submit your work via pull request or as directed by the project maintainers
Your contribution is valuable! Thank you for helping to expand the Advanced Tools platform.
